# ntutils

## Features

* Memory utility functions to reduce writing effort
* Advanced assert functions
* Test utility functions
* String type with short string optimization
* UTF-8 support
* Collection types

  * Linear list
  * Circular buffer
  * Queue (TODO)
  * Hash set
  * Hash map
  * Multi value hash map (TODO)

* JSON parser
* File system utility functions (TODO)
